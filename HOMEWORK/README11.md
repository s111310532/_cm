1.特徵方程式的轉化對於一個 $n$ 階微分方程 $a_n y^{(n)} + \dots + a_0 y = 0$，我們假設解的形式為 $y = e^{rx}$。
代入後可得到特徵方程式：$$a_n r^n + a_{n-1} r^{n-1} + \dots + a_0 = 0$$程式第一步使用 np.roots 就是在自動化求解這個高次方程式的所有 $r$。
2. 處理三種根的情況根據求出的根 $r$，通解的組成方式不同，程式中做了相對應的邏輯判斷：相異實根：每個根 $r$ 對應一個項 $C e^{rx}$。
重根（Repeated Roots）：如果 $r$ 出現了 $k$ 次，根據線性獨立原則，我們必須補上 $x, x^2, \dots, x^{k-1}$ 作為權重，即 $(C_1 + C_2 x + \dots)e^{rx}$。程式中透過 unique_roots 和 counts 來捕捉並處理這種情況。
複數根：若出現 $r = \alpha \pm \beta i$，利用歐拉公式，我們將解寫成 $e^{\alpha x}(C_1 \cos \beta x + C_2 \sin \beta x)$，這比直接寫複數指數更符合物理與工程上的實數解需求。
3. 數值穩定性在 np.unique(np.round(roots, 10)) 這一行中，我加入了 round。這是因為數值解（如 $1.9999999999$）在電腦中與 $2.0$ 可能被視為不同，若不處理會導致「重根」判定失敗。
